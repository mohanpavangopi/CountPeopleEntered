# 👥 People Counter Web App

This is a simple web application that allows users to count how many people have entered a place. It includes buttons to increment, decrement, and save the count.

## 🚀 Features

- ✅ Count people by clicking the "INCREMENT" button
- ✅ Decrease count using "DECREMENT"
- ✅ Save the current count to a log using "SAVE"
- ✅ Resets the count to 0 after saving

## 🛠️ Technologies Used

- HTML
- CSS (optional, for styling)
- JavaScript
